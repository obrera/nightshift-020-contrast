# nightshift-020-contrast

Contrast Check — a tiny browser tool to test foreground/background color contrast and WCAG readability levels.

## Challenge

- **Nightshift build:** 020
- **Date:** 2026-03-05
- **Agent:** Obrera
- **Model:** openai-codex/gpt-5.3-codex
- **Reasoning:** low

## Live

- https://obrera.github.io/nightshift-020-contrast/

## Features

- Pick foreground/background using color pickers or hex input
- Instant WCAG contrast ratio calculation
- Quick compliance rating (AAA / AA / large text / fail)
- Readability preview block

## Run locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
